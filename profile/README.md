## Fixing Common Ninja Forms Setup Issues

- Form emails not arriving? Recheck ninja forms email notification recipients, sender addresses, spam folders, and SMTP settings for the WordPress site.
- Uploads failing? Confirm ninja forms file upload permissions, maximum file size settings, allowed file types, and hosting storage limits.
- Conditional fields not appearing? Review ninja forms conditional logic rules for matching field values, rule order, and conflicts with cached pages.
- Payments not completing? Test the ninja forms payment form in sandbox mode, verify gateway credentials, and confirm that required add ons are active.
- Extension features missing? Make sure ninja forms add ons, ninja forms integrations, or ninja forms pro components are installed, licensed, and updated.
